# anchors/

`python main.py --calibrate-anchor` speichert hier:

- `table_anchor_patch.png` — Bild des gewählten Desktop-Rechtecks (nach Ausblendung des Overlays)
- `table_anchor_config.json` — Metadaten (`origin: top_left`, Bildschirm-Rechteck, …)

Diese Dateien sind **lokal** und werden nicht ins Repository committet.
